# Moni

Landing page de validación para Moni. Encuesta corta para freelancers monotributistas que se olvidan de los vencimientos de ARCA.

## Stack

- HTML / CSS / JavaScript vanilla
- FormSubmit para envío de respuestas por email
- Google Fonts: Syne + DM Mono

## Uso local

Abrir `moni-validacion.html.html` en el navegador.

## Email de contacto

Las respuestas de la encuesta se envían a `monivalidacion@gmail.com` via [FormSubmit](https://formsubmit.co).
